Multi-select

Multi-Select Component

Component
npm Package @sembilan/multi-select
Peer Dependencies lodash-es
bootstrap@next
@sembilan/bootstrap-theme

Description

Multi-Select Component show list of options and enable user to select multiple values. Should be used with ngModel.

Inputs

placeholder

string (default: "")

Shown in input as a hint. Hide when there's value selected.

options

array (default: [])

List of option selectable by user.

comparator

function (default: (a1: string, a2: T) => a1 === a2)

Custom comparator for comparing user written value with options. Accept function which accept first argument as string and second argument as object. Should return a boolean.
